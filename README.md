# Mapping the Broadway Network

## Overview
Constructing a performer collaboration network from Broadway production data.

## Data
Source: IBDB

## Pipeline
1. Scrape productions
2. Extract opening night casts
3. Construct bipartite graph
4. Project into performer network
5. Analyse network properties
6. Visualise interactively

## Current status
- 7,058 productions
- 55,263 performers
- 1.5M collaboration edges
